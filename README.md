# Vibe Match

## Project Description
Vibe Match is a web app that recommends music based on the user's mood. Users will be able to select how they are feeling and receive song suggestions that match that mood. The goal of the app is to make discovering music that fits the moment simple and enjoyable.

## Planned Features
- Select a mood - users will be able to select a mood from a list of options
- View song recommendations based on that mood
- Shuffle or generate new song suggestions
- Save favorite songs that they liked
- View a list of the saved favorites
